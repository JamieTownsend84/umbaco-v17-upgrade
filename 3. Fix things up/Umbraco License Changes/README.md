## Umbraco License Changes - [Examples](./examples.md)

For a while Umbraco have advised their licensing is changing to align all of their products to a subscription model. This change mostly affects Umbraco forms where most clients had a one-off license by installing a `.lic` file to a specific folder. This has now changed and clients will need to delete these files and move to the new subscription model ASAP.

There is lots of information on this blog post regarding how clients update their licenses. \
https://umbraco.com/blog/umbraco-17-lts-release/

Along with this pricing model update, how you configure the license has also changed. You no longer save a `.lic` file - If you've worked with Umbraco Engage or another product you'll be familiar with a license key, *However*, the appsetting path these are placed in has changed also, see [examples](./examples.md).
