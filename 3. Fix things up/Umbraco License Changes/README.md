## Umbraco Licence Changes - [Examples](./examples.md)

For a while, Umbraco have advised their licensing is changing to align all of their products to a subscription model. This change mostly affects Umbraco Forms, where most clients had a one-off licence by installing a `.lic` file to a specific folder. This has now changed and clients will need to delete these files and move to the new subscription model as soon as possible.

There is plenty of information in this blog post regarding how clients update their licences. \
https://umbraco.com/blog/umbraco-17-lts-release/

Along with this pricing model update, how you configure the licence has also changed. You no longer save a `.lic` file - if you've worked with Umbraco Engage or another product you'll be familiar with a licence key; however, the appsettings path these are placed in has also changed, see [examples](./examples.md).
