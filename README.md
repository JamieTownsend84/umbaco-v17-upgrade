# A Developer's Guide

### Jamie Townsend
Principal Full-Stack Engineer \
jamie.townsend@shoutdigital.com

---

With Umbraco 17 now the current Long-Term Support release, many organisations are planning their migration from version 13. This talk goes beyond documentation to address real-world upgrade challenges: adapting custom extensions for the new architecture, handling breaking changes in .NET 10, and writing code migrations for packages that don't have an automatic upgrade path. Includes worked examples and reusable patterns.
